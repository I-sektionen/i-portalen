Bakgrund och generell info

Evenemang kan skapas av både privatpersoner och organisationer.

Skapa ett evenemang

Inloggade användare kan klicka på Innehåll i menyn och sedan på Skapa ett event. Fyll i text och datum enligt instruktionerna, välj om en organisation står för eventet och välj eventuella taggar. Notera att eventet måste godkännas av administratör innan publicering även om datumet för publicering är passerat. Du kan även klicka på fliken förhandsvisning för att se hur eventet kommer att se ut när det är publicerat. Du kan också bifoga filer. Du får ett mail när eventet blivit godkänd eller avslaget.

Administrera evenemang

Behöriga användare kan klicka på Administrera evenemang när ett event har valts.

Incheckning

Klicka på Starta incheckning. Fyll i besökarens Liu-ID och klicka på Checka in, då visas om besökaren är anmäld eller ej, och även om den ska få mat. Besökarens nummer till talarlistan visas också. Om kortläsare (RFID-läsare) används för incheckningen så fungerar den som ett tangentbord och skriver in personens RFID och sedan på Checka in automatiskt.

Talarlista

När deltagarna checkar in får de automatiskt ett talarnummer. Dessa listas under länken Se lista över talarnummer i adminvyn för eventet. Under Admin Talarlistan kan en administratör lägga till/ta bort talare till talarlistan genom att fylla i talarnumret från incheckningen, växla till nästa talare samt rensa talarlistan. Under Se Talarlistan visas sedan nuvarande talarlista.

Ändra ett evenemang

Om du har skapat ett event så kan du klicka på Mina event under menyknappen Innehåll. Där kan du klicka på Ändra för att redigera ditt event. Den nya ändringen måste godkännas av administratör innan den publiceras. Den gamla versionen ligger kvar under tiden om den tidigare versionen redan blivit godkänd. Du får ett mail när eventet blivit godkänt eller avslaget.

Man kan även redigera ett event ifrån adminvyn under Ändra event.

Mina evenemang

Under menyknappen Innehåll finns en länk till Mina Event där du ser dina event och status för dessa. Statusen kan vara "Godkänt" (Eventet är publicerat), "Väntar på godkännande" (En administratör behöver godkänna eventet innan det publiceras) eller "Avslaget" (Eventet i nuvarande form avslogs, och anledningen till detta specifieras).

Godkänn ett evenemang

En administratör hittar alternativet Godkänn Event under menyknappen Innehåll. Där listas alla event som väntar på godkännande. Det finns tre alternativ för varje event, Godkänn, Avslå eller Gör ändringar. Ett mail skickas ut till skaparen när statusen ändrats.

Ta bort ett evenemang

Du kan inte "ta bort" ett evenemang i den bemärkelsen att den försvinner ur listan över dina evenemang. När tiden för publiceringen har gått ut så kommer evenemanget däremot inte att vara tillgänglig för någon annan vilket gör att evenemang inte behöver tas bort.

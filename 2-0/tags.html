Hantera taggar i Strapi

Innehållsförteckning

  • Skapa taggar
  • Redigera taggar
  • Radera taggar

  1. Logga in på Strapi.
    1. Notera: För att kunna hantera utskott/sektionssidor behöver du ha kontobehörighet som en Editor (dvs. utskottsordföranden, m.m.) eller Super Admin.
  2. Klicka på “Content Manager” i menyn till vänster.
  3. Klicka på “tag”.

Skapa taggar

  1. Klicka på “Create new entry” i högre högra hörnet.
  2. Fyll i information i alla obligatoriska fält.
  3. Klicka på “Spara” i det övre högra hörnet.

Redigera taggar

  1. Klicka på det tag-elementet som ska redigeras.
  2. Gör ändringarna.
  3. Klicka på “Spara” i det övre högra hörnet.

Radera taggar

  1. Leta fram bokningen som ska raderas genom att söka, filtrera, sortera eller scrolla i listan.
  2. Klicka antingen:
    1. direkt på “radera” till höger i listan för det element som ska raderas.
    2. på taggen som ska raderas, och klicka sedan på “Delete this entry”.

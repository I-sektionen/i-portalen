Logga in på I-portalen som användare

  1. Klicka på “LOGGA IN” uppe till höger.
  2. Klicka på “Logga in på I-Portalen via LiU”.
  3. Scrolla längst ner och klicka på “Fortsätt med LiU”.
  4. Klicka på “Active Directory”.
  5. Logga in med din LiU-inloggning

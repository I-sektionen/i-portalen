Hur man skapar och hanterar ett möte (sektionsmöte)

På i-portalen går det att skapa vad som kallas Möten. Detta är en variant av evenemang där det går att ha en talarlista och omröstningar. Möten skapas via Strapi och hanteras sedan via i-portalen/Strapi.

Notera: För att kunna hantera sektionsmöten på i-portalen behöver du ha admin-behörigheter. I huvudsak är det endast Webgroup och Styret som har denna behörighet.

Skapa möte

  1. Logga in på Strapi.
  2. Klicka på “Content Manager” i menyn till vänster.
  3. Klicka på “Meeting” i content-menyn som dyker upp.
  4. Klicka på “Create new entry” uppe till höger.
  5. Ange namn, beskrivning samt start- och sluttid. Se här hur formateringen i textrutor fungerar.
    1. Notera att fältet “Views” inte behöver fyllas i. Detta fält visar hur många visningar ett möte har efter att det har publicerats.
  6. Lägg till plats genom att trycka på “position” och ange namnet på platsen/salen.
  7. Lägg till “relations” till höger för att ange vem som skapat eventet. Under “association” finns alla utskott och under “group” resterande tillhörande sektionen. Det går även att lägga till taggar under fältet “tags”.
  8. Om du vill lägga till anmälda personer till eventet redan nu går det att göra det i fältet “users”.
  9. Klicka på “Save” uppe i högra hörnet.

Redigera möte

  1. Logga in på Strapi.
  2. Klicka på “Content Manager” i menyn till vänster.
  3. Klicka på “Meeting” i content-menyn som dyker upp.
  4. Leta reda på det möte du vill redigera i listan som visas. Det går att sortera, filtrera eller söka i listan för att enklare hitta det sökta objektet.
  5. Klicka på objektet du vill redigera.
  6. Redigera de fält som ska ändras. Se här hur formateringen i textrutor fungerar.
    1. OBS: Redigera inte fältet “eventid”.
  7. Klicka på “Spara” i övre högra hörnet.

Radera möte

  1. Logga in på Strapi.
  2. Klicka på “Content Manager” i menyn till vänster.
  3. Klicka på “Meeting” i content-menyn som dyker upp.
  4. Leta reda på det möte du vill redigera i listan som visas. Det går att sortera, filtrera eller söka i listan för att enklare hitta det sökta objektet.
  5. Klicka antingen:
    1. Direkt på “radera” till höger i listan för det element som ska raderas.
    2. På bokningen som ska raderas, och klicka sedan på “Delete this entry”.

Starta ett möte

  1. Klicka på “Admin panel” på i-portalen till höger i huvudmenyn.
  2. Leta efter mötet du vill starta i listan “Uppkommande möten”.
  3. Klicka på “Starta”. Listan ska nu även ha lagts till i listan “Aktiva möten”.

Checka in anmälda

  1. Klicka på “Admin panel” på i-portalen.se till höger i huvudmenyn.
  2. Leta efter mötet du vill checka in personer till i listan “Aktiva möten”.
  3. Klicka på knappen “Kontrollpanel”.
  4. Klicka på “Öppna användare”.
  5. Gå antingen igenom listan eller sök på id, LiU-id eller namn för att hitta det namn som ska checkas in till mötet.
    1. Notera: Sökfunktionen är skiftlägeskänslig (case sensitive).
  6. Klicka på “Närvarande”.
    1. Notera: Om knappen klickas på igen kommer personen att markeras som inte incheckad igen.
  7. Klicka på “Stäng användare” om du vill sluta visa listan.

Talarlista

  1. Klicka på “Admin panel” på i-portalen.se till höger i huvudmenyn.
  2. Leta efter mötet du vill checka in personer till i listan “Aktiva möten”.
  3. Klicka på knappen “Kontrollpanel”. Talarlistan kommer nu att visas i ett popup-fönster.
  4. Skriv in id, LiU-id eller namn i sökfältet och klicka sedan på “Lägg till talare i talarlistan” för att lägga till användare i talarlistan.
  5. Klicka på krysset för att ta bort användare från talarlistan som visas bakom namnet.
  6. Klicka på “Visa talarlista på egen sida” för att visa en egen webbsida med endast talarlistan på.

Röstningssystem

  1. Klicka på “Admin panel” på i-portalen.se till höger i huvudmenyn.
  2. Leta efter mötet du vill checka in personer till i listan “Aktiva möten”.
  3. Klicka på knappen “Kontrollpanel”. Talarlistan kommer nu att visas i ett popup-fönster.
  4. Klicka på “Omröstning”. Om det redan pågår en omröstning kommer denna att visas till vänster på sidan som visas.
  5. För att skapa en ny omröstning fyller du i fälten till vänster under “Starta omröstning?”. Om det redan pågår en omröstning kommer denna att försvinna.
  6. Klicka på “Stoppa omröstning” för att avsluta en omröstning. Resultaten från omröstningen kommer inte att sparas.

Avsluta ett möte

  1. Klicka på “Admin panel” på i-portalen.se till höger i huvudmenyn.
  2. Leta efter mötet du vill checka in personer till i listan “Aktiva möten”.
  3. Klicka på knappen “Kontrollpanel”.
  4. Klicka på “Avsluta möte” för att avsluta det pågående mötet.
